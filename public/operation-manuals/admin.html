1. 工作台概览

管理员端用于维护虚拟仿真实验运行所需的基础资源。左侧菜单包含实验手册管理、实验数据管理、用户管理、班级管理和学年学期管理。

建议先配置学年学期，再创建教师、助教、学生和班级，最后上传实验手册与实验数据。

2. 实验手册管理

  1. 进入“实验手册管理”。
  2. 点击新增按钮，填写手册名称和备注，上传 PDF 文件。
  3. 上传后可在列表中编辑名称、备注，或删除不再使用的手册。
  4. 通过启用开关设置当前有效手册。系统同一时间只应保留一份启用手册。

学生端实验介绍页会读取当前启用的手册。如果学生看到空手册或旧手册，请先检查此处是否已正确启用。

3. 实验数据管理

实验数据决定学生可选择的行业、企业、产品和销量序列。上传前请保证 CSV 表头与系统要求一致。

字段说明
行业名称学生第一步选择的行业。
公司名称行业下可选企业。
产品名称企业下可选产品。
年份、月份组成历史销量时间序列。
销售数量、数量单位用于模型训练、评估和报告展示。
上传新数据后，正在实验中的学生不会自动回滚流程。若需要统一使用新数据，请在教学安排中明确重新开始实验的时间。

4. 用户管理

教师与助教

  • 可单个添加教师或助教，填写账号、姓名、邮箱、手机号和初始密码。
  • 可批量添加教师或助教，系统会按当前规则生成账号信息。
  • 可对非当前管理员账号执行重置密码、编辑和删除等操作。

学生

  • 学生通常通过班级导入或批量创建进入系统。
  • 删除学生前，请确认该学生的实验记录、报告和成绩是否仍需保留。

5. 班级管理

  1. 进入“班级管理”，创建班级并绑定学年学期。
  2. 为班级分配教师。教师登录后只管理自己负责的班级。
  3. 可为班级分配助教，助教仅能访问被授权的班级。
  4. 维护班级学生名单，保证实验进度和成绩统计归属正确。

6. 学年学期管理

学年学期用于区分不同教学周期。创建班级、查看统计和分配教学资源时，应选择正确的学期。

  • 新增学期前，请确认名称和时间范围。
  • 当前学期会影响教师端默认查看的数据范围。
  • 历史学期数据一般用于追溯，不建议随意删除。

7. 账户与安全

  • 点击右上角头像，可选择“修改密码”或“退出登录”。
  • 修改密码时需要输入当前密码，并确认新密码。
  • 公共电脑使用完毕后必须退出登录。

8. 常见问题

学生看不到实验手册

检查“实验手册管理”中是否存在已启用的 PDF 手册。

教师看不到班级

检查班级是否绑定到该教师，并确认教师查看的学年学期正确。

助教无法进入考核管理

助教需要先被分配可管理班级。没有授权班级时，部分考核入口会被隐藏或跳转。
