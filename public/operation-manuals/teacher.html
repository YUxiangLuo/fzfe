1. 工作台概览

教师端用于跟踪学生实验进度、评阅实验报告、维护题库和成绩规则。左侧菜单分为实验管理、班级管理、学生管理、考核管理和账户设置。

如果页面数据为空，先检查右上或页面中的学年学期选择是否正确，再确认管理员是否已将班级分配给当前教师。

2. 实验管理

实验进度

  • 查看班级学生的实验流程完成情况。
  • 关注长时间停留在早期步骤或训练步骤失败的学生。
  • 可结合实验日志定位学生最后一次操作和异常信息。

实验报告

  1. 进入“实验报告”，选择班级和筛选条件。
  2. 打开学生报告，预览或下载 PDF。
  3. 填写实验报告得分和反馈意见。
  4. 需要学生重做时，选择驳回并写清原因。
驳回报告会让学生看到反馈并重新进入实验流程。请在反馈中说明需要补充或修正的内容。

实验日志

实验日志用于查看学生在关键步骤的操作记录，适合排查无法继续、结果异常或报告缺失等问题。

3. 班级与学生

班级管理

  • 查看本人负责的班级及班级助教。
  • 确认班级所属学期和学生人数。
  • 需要新增、删除或调整班级归属时，联系管理员处理。

学生管理

  • 查看班级内学生账号、姓名和基础信息。
  • 按班级筛选学生，核对学生是否已正确加入教学班。
  • 学生忘记密码或账号异常时，按学校流程联系管理员或执行可用的重置操作。

4. 考核管理

页面用途
题库管理维护预测模型和生产计划相关测验题。
成绩权重设置实验流程、知识测验、模型质量和报告质量等成绩占比。
成绩总览按班级查看学生最终成绩、报告状态和组成项。

修改题库或成绩权重前，应确认是否会影响当前学期正在进行的教学评价。

5. 助教管理

  1. 进入“账户设置 / 助教管理”。
  2. 可创建新助教，也可选择已有助教。
  3. 为助教分配可管理班级。
  4. 需要变更权限时，重新分配或移除对应班级。

助教只能访问被分配的班级。教师应定期检查助教班级权限是否与教学安排一致。

6. 个人信息与密码

  • 进入“账户设置 / 个人信息”查看账号信息。
  • 在个人信息页修改密码，按提示输入当前密码、新密码和确认密码。
  • 右上角头像菜单提供退出登录入口。

7. 推荐教学流程

  1. 课前确认班级、学生名单、助教权限和题库配置。
  2. 实验开始后，通过实验进度观察学生是否顺利推进。
  3. 学生提交报告后，集中进行报告评阅和反馈。
  4. 评分完成后，在成绩总览中核对最终成绩和各项组成。

8. 常见问题

看不到某个班级

确认当前学期是否正确，并联系管理员检查班级是否分配给当前教师。

报告 PDF 无法打开

刷新报告列表后重试。若系统提示文件缺失，请联系管理员检查服务器报告文件。

助教看不到考核相关页面

助教需要拥有至少一个可管理班级。请在助教管理中检查班级分配。
